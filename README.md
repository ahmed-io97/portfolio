# Taukir Ahmmed — Portfolio

A sleek, modern portfolio website showcasing my work as a software engineer and cybersecurity specialist.

**Live Demo:** [https://ahmed-io97.github.io/portfolio](https://ahmed-io97.github.io/portfolio)

---

## 🛠 Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Modern styling with custom variables & animations
- **JavaScript** — Interactive features (smooth scrolling, copy-to-clipboard, custom cursor)
- **GitHub Pages** — Hosting

---

## ✨ Features

- Clean, professional design with a modern layout
- Fully responsive (mobile, tablet, desktop)
- Smooth navigation and animations
- Dark/modern color scheme
- Copy-to-clipboard email functionality
- Quick links to GitHub & LinkedIn

---

## 📚 Sections

1. **About** — Bio and professional summary
2. **Experience** — Work history, internships, and education
3. **Projects** — Featured portfolio projects with tech stacks
4. **Skills** — Core competencies across development & cybersecurity
5. **Certifications** — Professional certifications
6. **Contact** — Email, LinkedIn, and GitHub links

---

## 🚀 How to Use

### View Online
Open [https://ahmed-io97.github.io/portfolio](https://ahmed-io97.github.io/portfolio) in your browser.

### Run Locally
1. Clone the repo:
   ```bash
   git clone https://github.com/ahmed-io97/portfolio.git
   cd portfolio
   ```
2. Open `index.html` in your browser (no server needed)
